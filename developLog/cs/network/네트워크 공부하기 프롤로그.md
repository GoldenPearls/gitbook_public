# 네트워크 공부하기 프롤로그

![](https://velog.velcdn.com/images/prettylee620/post/d74d2a04-b312-4dca-ac8c-5041ac85f124/image.png)

**이번달에 일없을 때나 집에서 공부해서 네트워크 뿌신다!!!**
